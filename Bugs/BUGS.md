## Bug-1
**Заголовок:** Ошибка в фильтре "Диапазон цен": найдена цена выше максимальной (40000)

**Приоритет:** Major  
**Серьёзность:** High  
**Окружение:** Chromium 112, Windows 10, Dev (протестировано на стенде)

**Шаги воспроизведения:**
1. Открыть главную страницу сайта.
2. Перейти в фильтр "Диапазон цен".
3. Задать диапазон цен от 0 до 40000.
4. Нажать на кнопку "Применить".
5. Проверить, что все цены на странице находятся в пределах заданного диапазона (до 40000).

**Фактический результат (Actual Result):**
- На странице отображается товар с ценой, превышающей 40000 (например, 44518).

**Ожидаемый результат (Expected Result):**
- Все цены на странице должны быть не выше 40000.

## Bug-4
**Заголовок:** Таймер не запускается после паузы на странице статистики

**Приоритет:** Major  
**Серьёзность:** Medium  
**Окружение:** Chromium 112, Windows 10, Dev (протестировано на стенде)

**Шаги воспроизведения:**
1. Открыть главную страницу сайта.
2. Перейти на страницу статистики.
3. Остановить таймер с помощью кнопки остановки.
4. Нажать кнопку для запуска таймера.
5. Проверить, что таймер запускается.

**Фактический результат (Actual Result):**
- Таймер не запускается, кнопка "Отключить автообновление" не становится видимой.

**Ожидаемый результат (Expected Result):**
- После нажатия кнопки запуска таймера, он должен стать активным, а кнопка "Отключить автообновление" должна стать видимой.
